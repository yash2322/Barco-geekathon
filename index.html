Emergency Vehicle

By recognizing vehicle number plate using ALPR (Automatic License Plate Recognition) API and extracting vehicle number from the number plate, we will able to detect ambulance on the road(by quiring the number in the database of ambulance in the city) .

Stolen Car

Owner of stolen car will FIR about the car and put the vehicle number in our portal. In stolen car data base this new data will be added and after extracting vehicle number through ALPR API and OCR API system will detect the vehicle on the street. If it is so then traffic police will take charge to catch the thief.
